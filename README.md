# Samiya Wedding Center

## Requirement
  - Node.js
  - bun

## Installation
Follow these steps:

```sh
git clone https://github.com/anshifmonzz/samiya-wedding-center.git
cd samiya-wedding-center
bun i
bun run dev
```

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
